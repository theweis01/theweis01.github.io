Welcome to the SAS Studio Repository Toolkit Samples

The following sample repositories are available.

  • Simple Repository simple

    The simple repository contains the minimal repository definition for a single task category and single snippet category. The only elements in the repository XML files are the required tags. If you've never written a repository this is where to start.

  • Snippets Repository snippets

    This repository has several snippet categories defined with SAS code taken from the SAS Knowledge Base and Reference on http://support.sas.com. If you need to provide SAS samples or fragments of SAS code to others in your organization this repository is a good place to start.

    In addition to showing several snippet categories this repository demonstrations these additional repository features.

    1. Specifying an icon for the repository that resides external to the repository.
    2. Specifying an icon for individual snippet that is contained within the repository.
    3. Annotating the repository with documentation that is stored in the repository.
    4. Annotating a snippet category with documentation not stored in the repository.
    5. Annotating individual snippets with documentation not stored in the repository.
    6. Referencing data files stored in the repository (See the Import Category) - This approach assumes that the SAS server can access URLs where the repository resides by leveraging the URL Access Method provided by the SAS language. The SAS code is co-located with external file and resolved to a complete URL when accessed.

  • Macro Repository macros

    This repository has a snippet and task using SAS macros referenced from the repository. It illustrates how macros can be packaged within a repository and how the URL Access Method can be used to %INCLUDE the repository macros.

  • Saved Task Repository tasks

    This repository has only task models (CTM) and saved tasks (CTK). The default List Data CTM file is in the repository along with the following CTK variants:

    1. List Data - SASHELP.FISH along with an external repository icon to display.
    2. List Data - SASHELP.GAS
    3. List Data - SASHELP.VOPTIONS with only two columns selected. An internal repository icon should display and an external NESUG paper entitled "Introduction to SAS Dictionary Tables".

    To predefine all the example repositories the following repositories.xml can be put into global preferences directory. 

    
<?xml version="1.0" encoding="UTF-8"?>
<Repositories>
<Repository uri="snippets" name="Snippet Repository" />
</Repositories>
